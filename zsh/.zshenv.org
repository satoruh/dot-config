https://specifications.freedesktop.org/basedir-spec/basedir-spec-latest.html

** mise

mise はデフォルトで XDG に従う(data = =${XDG_DATA_HOME}/mise=、config =
=${XDG_CONFIG_HOME}/mise=)ため、環境変数の明示は不要。

非対話シェルからも参照できるよう shims を PATH に加える。対話シェルでの
=mise activate= は =.zshrc= 側で行う。

#+begin_src zsh
path=(
    ${XDG_DATA_HOME}/mise/shims(N-/)
    $path
)
#+end_src

** Homebrew

#+begin_src zsh

  )
fi
